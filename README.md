# 灵枢平台文档

本仓库沉淀灵枢平台第一阶段的产品、架构、契约、验收和 Alpha 开工文档。

## 重点入口

- [PRD](docs/PRD.md)：Alpha 报告类数字商品主闭环的产品需求。
- [TODO](docs/TODO.md)：Alpha 开工、排期和验收用待办清单。
- [文档索引](docs/README.md)：全部设计文档的阅读顺序和维护规则。

## 当前范围

Alpha 聚焦 `research_report` 报告类数字商品主闭环：

```text
供端能力可调用
  -> 产端执行报告线 Workflow
  -> 生成正式 ProductPackage
  -> 销端上架、预览、模拟购买、下载
  -> 生成绑定版本的 MarketFeedback
```

暂不进入 Alpha 的能力包括真实支付、自动优化任务、视频生产线、完整运营仪表盘、创作者店铺和外部开放 API。
